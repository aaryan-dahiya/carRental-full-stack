# Car Rental MERN Project

A full-stack car rental application built with MERN stack and AI integration.

## Features
- Browse cars
- Check availability
- AI-powered chatbot for car features
- Booking system

## Deployment
Check the live project here: [Car Rental Live Demo](https://car-rental-peach-eight.vercel.app/)

## Technologies Used
- MongoDB, Express, React, Node.js
- AI Integration (Groq)
